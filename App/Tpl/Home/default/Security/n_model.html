扣除以上特殊原因外，常态化的航运时间会依据不同国家、不同港口所需要的海运时间是基本稳定的，因为海运主要都隶属于全球知名的大船公司，以及特殊情况出现的概率、周期性、处理特殊事件预计的时间等，整体时间仍然是相对可以控制在一定的时间范围内，我们可以称之为“相对确定时间”。
是木材质押段，标期取决于借款人的实际需求及链金所的风险控制（货物价格调整周期等），考虑到木材境内段可能会存在销售周期，链金所会在提单质押的基础上明确一个现货质押的时间，但是因为现货在质押的同时，也会进行销售，如果货物质押后，在质押时间内找到买家，借款人就会先行还款，并将货物提走，但货物的销售时间也是相对不固定的，到借款人还款提货的时候，标的结束。
所以，后一个N也是在固定期限之内的相对不确定，有一定的变化空间。“N+N”模式体现了链金所的“产融结合”的模式的设计理念，是对整个进口木材行业经营规则深切了解的应用，是对产业链内的收付、结算及风险的全方位理解，链金所不搞“一刀切”模式，一户一策的支持木材产业链内的中小微企业的发展。
